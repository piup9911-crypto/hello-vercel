聊天记录面板入口

这里是你网页里的可见入口。真正的聊天记录仍然保存在电脑本地，所以要先启动本地 agent， 然后从这里打开本机面板来查看、删除、归档 Telegram/Gem CLI 记录。

打开本机聊天记录面板 查看说明文档 查看工具桥
先启动本地 agent

在电脑上运行启动脚本，网页才能读到本地聊天 JSON。

tools\gemini-cli-telegram\start-gem-chat-record-manager.cmd
再打开面板

本机浏览器打开这个地址，才能进行查看、删除、归档和手机预览。

http://127.0.0.1:4144/
不会上传私密记录

GitHub/Vercel 只保存入口和源码，不保存你的真实聊天记录、token 或运行日志。

bridge-state/ 不提交

为什么不能直接在 Vercel 页面里显示记录？

Vercel 服务器访问不到你电脑里的 bridge-state\chats。 所以这个页面负责做入口和说明；真正的数据面板由你电脑上的本地 agent 提供。
